# Niger State SMC Performance Dashboard

## Overview
This project presents a Seasonal Malaria Chemoprevention (SMC) Performance Dashboard developed using Power BI with a hypothetical dataset from Niger State, Nigeria.

The dashboard was designed to analyze and visualize key program performance indicators to support data-driven decision-making within public health programs and Monitoring & Evaluation (M&E) activities.

---

## Features
The dashboard focuses on key SMC performance indicators, including:

- Coverage Rate
- Adherence Rate
- Adverse Event Rate
- Missed Children Analysis
- LGA-Level Performance Comparison
- Interactive Filtering and Trend Analysis

---

## Key Insights / Project Outcomes
- Coverage exceeded 80% across all LGAs
- Adherence remained consistent across implementation cycles
- Missed children were identified across multiple LGAs, highlighting the need for targeted follow-up strategies
- One LGA recorded a 7% adverse event rate, requiring further investigation and monitoring

---

## Tools & Techniques Used
- Power BI
- DAX Measures & Calculated Columns
- Data Modeling
- Interactive Slicers
- KPI Cards
- Data Visualization Techniques

---

## Purpose of the Project
This dashboard was developed as a practical M&E and data analytics project to simulate real-world program monitoring and performance assessment within public health interventions.

The project demonstrates how data visualization and analytics can support:
- Program monitoring
- Evidence-based decision-making
- Identification of service delivery gaps
- Improved reporting and accountability
